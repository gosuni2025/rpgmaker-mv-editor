# 배포

게임을 완성한 뒤 **파일 → 배포** 메뉴에서 다양한 플랫폼으로 배포할 수 있습니다.

![배포 다이얼로그](images/deploy-dialog.png)

## 배포 대상

| 탭 | 설명 |
|------|------|
| **itch.io** | butler CLI로 itch.io에 직접 업로드 |
| **Netlify** | Netlify 사이트에 배포 |
| **GitHub Pages** | GitHub Pages 브랜치에 push |
| **로컬 폴더** | ZIP 또는 폴더로 내보내기 |

## itch.io 배포

itch.io 탭에서는 butler CLI를 통해 게임을 바로 업로드합니다.

- **사전 조건**: butler 설치 및 로그인 필요
- **Username / 프로젝트**: itch.io 계정과 프로젝트 URL 설정
- **채널**: 배포 채널 (HTML5 게임은 `html5`)
- **새 게임 만들기**: 프로젝트가 없으면 itch.io에 자동 생성

## 배포 옵션

### SW 번들링

이미지, 오디오, 데이터를 ZIP으로 번들링합니다. 브라우저가 Service Worker로 번들을 캐싱하여 로딩 속도를 개선합니다.

### 캐시 버스팅

소스코드, 이미지, 오디오, 동영상, 데이터 카테고리별로 캐시 버스팅을 선택할 수 있습니다. 업데이트된 리소스가 브라우저 캐시 없이 즉시 반영됩니다.

### PNG → WebP 변환

이미지를 WebP 포맷으로 자동 변환하여 용량을 줄입니다.

### 미사용 에셋 제외

프로젝트에서 실제로 사용하지 않는 이미지·오디오 파일을 배포에서 제외하여 용량을 최소화합니다.
