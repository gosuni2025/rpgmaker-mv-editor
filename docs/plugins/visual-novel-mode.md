# VisualNovelMode — 비주얼 노벨 모드

**파일**: `js/plugins/VisualNovelMode.js`
**버전**: v1.1

## 기능 요약

게임 메시지 표시를 비주얼 노벨(VN) 스타일로 변경합니다. 화면 하단이 아닌 **화면 전체를 덮는 오버레이** 영역에 텍스트가 타이프라이터 효과로 출력됩니다. 이전 대화 내용이 누적되며 선택지를 인라인으로 표시합니다.

## 주요 특징

- **타이프라이터 효과** — 문자가 하나씩 출력
- **누적 텍스트** — 이전 메시지가 위에 쌓이며 스크롤
- **인라인 선택지** — 텍스트 아래에 선택지가 자연스럽게 연결
- **화자명 강조** — 발화자 이름을 별도 색상으로 표시
- **자동 진행** — 일정 시간 후 자동 탈출 옵션

## 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| overlayOpacity | 120 | 배경 어두워짐 정도 (0~255) |
| transitionFrames | 24 | 화면 전환 프레임 수 |
| textAreaX | 80 | 텍스트 영역 X 위치 |
| textAreaY | 300 | 텍스트 영역 Y 위치 |
| textAreaWidth | 584 | 텍스트 영역 너비 |
| textAreaHeight | 400 | 텍스트 영역 높이 |
| choiceStyle | inline | 선택지 스타일 (default / inline) |
| speakerColor | #ffe066 | 화자명 색상 |
| choiceIndicator | ">" | 선택지 커서 문자 |
| autoExitDelay | 30 | 자동 탈출 딜레이 (프레임, 0=비활성) |
| showScrollBar | false | 스크롤바 표시 여부 |

## 플러그인 커맨드

| 커맨드 | 인자 | 설명 |
|--------|------|------|
| `VN enter` | — | VN 모드 시작 |
| `VN exit` | — | VN 모드 종료 |
| `VN choiceStyle` | `default` / `inline` | 선택지 스타일 변경 |

## 이벤트 커맨드 예시

```
[플러그인 커맨드] VN enter
[문장 표시] 에리카: 안녕하세요. 저는 에리카입니다.
[문장 표시] 에리카: 무엇을 도와드릴까요?
[선택지 표시]
  [도전하다]
  [돌아가다]
[플러그인 커맨드] VN exit
```

## 화자명 표시

메시지 첫 줄을 `이름:` 형식으로 작성하면 화자명이 `speakerColor`로 강조됩니다.

```
문장 표시:
에리카: 이것이 화자명입니다.
```

## 주의사항

- VN 모드 진입/탈출 시 화면 트랜지션이 실행됩니다.
- 기본 메시지 창 표시 중 VN 커맨드를 실행하면 충돌이 발생할 수 있습니다.
