# SkyBox — 파노라마 스카이박스

**파일**: `js/plugins/SkyBox.js`
**용도**: 3D 모드 전용

## 기능 요약

3D 모드에서 Equirectangular(360°) 파노라마 이미지를 스카이 돔에 매핑하여 하늘 배경을 표시합니다.
카메라가 회전하면 스카이박스도 함께 회전하여 실감나는 원근감을 연출합니다.

## 설정 방법

### 1. 이미지 배치

파노라마 이미지를 프로젝트의 `img/skybox/` 폴더에 배치합니다.

지원 형식: PNG (권장: 4096×2048px 이상의 Equirectangular 파노라마)

### 2. 맵 인스펙터에서 설정

1. 맵 인스펙터 → **면 배경** 섹션
2. **Sky Sphere** 버튼 클릭 (Parallax에서 전환)
3. 텍스트 필드에 파일명 입력 (확장자 제외)

```
예: img/skybox/Panorama_Sky_15-512x512.png → "Panorama_Sky_15-512x512"
```

### 3. 회전 애니메이션 (선택)

스카이박스에 자동 회전 애니메이션을 적용하려면 이벤트 커맨드로 속도를 설정합니다.

## 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| Skybox Folder | skybox | 스카이 이미지 폴더 (`img/` 하위 경로) |
| Panorama File | sky_panorama | 기본 파노라마 파일명 |

## 이미지 규격 권장사항

| 항목 | 권장 |
|------|------|
| 형식 | Equirectangular (위도·경도 투영) |
| 해상도 | 2048×1024 이상 (4096×2048 최적) |
| 파일 형식 | PNG |

무료 HDRI/파노라마 이미지는 [Poly Haven](https://polyhaven.com/hdris) 등에서 구할 수 있습니다. Equirectangular PNG로 변환하여 사용하세요.

## 주의사항

- **3D 모드 전용** — 2D 모드에서는 표시되지 않습니다.
- Mode3D의 Y-flip 보정이 적용되어 있으므로 이미지가 상하 반전되어 보이면 정상입니다.
