# ItemBook — 아이템 도감

**파일**: `js/plugins/ItemBook.js`

## 기능 요약

플레이어가 게임 중 획득한 아이템, 무기, 방어구를 도감 형식으로 기록하고 열람합니다. 아직 획득하지 않은 아이템은 `???`로 표시됩니다.

## 주요 기능

- **아이템 / 무기 / 방어구** 모두 지원
- **획득 기록** — 한 번이라도 획득하면 영구 등록
- **미확인 표시** — 미획득 아이템은 `???`로 표시
- **아이콘 표시** — 각 항목의 아이콘 이미지
- **설명 표시** — 아이템 설명 텍스트

## 플러그인 커맨드

| 커맨드 | 설명 |
|--------|------|
| `ItemBook open` | 아이템 도감 열기 |
| `ItemBook add [ID]` | 특정 아이템 수동 등록 (획득 없이) |
| `ItemBook remove [ID]` | 특정 아이템 등록 해제 |

## 이벤트 커맨드 예시

```
[플러그인 커맨드] ItemBook open
```

메뉴 커맨드에 도감을 추가하려면 `Scene_Menu`를 수정하거나 커스텀 씬 엔진을 활용하세요.
