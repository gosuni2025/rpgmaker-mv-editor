# EnemyBook — 적 도감

**파일**: `js/plugins/EnemyBook.js`

## 기능 요약

전투에서 만난 적을 자동으로 도감에 기록하고, 메뉴에서 열람할 수 있습니다. 아직 조우하지 않은 적은 `??????`로 표시됩니다.

## 주요 기능

- **자동 기록** — 전투에서 조우한 적이 자동으로 등록
- **미확인 표시** — 미등록 적은 `??????`로 표시
- **커스텀 설명** — 적 메모 태그로 추가 설명 작성 가능
- **메뉴 통합** — 기본 메뉴에 "몬스터 도감" 항목 자동 추가

## 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| Unknown Data | `??????` | 미확인 적 표시 텍스트 |
| Show In Menu | `true` | 메뉴에 도감 항목 표시 여부 |
| Menu Text | `몬스터 도감` | 메뉴에 표시할 이름 |

## 플러그인 커맨드

| 커맨드 | 설명 |
|--------|------|
| `EnemyBook open` | 적 도감 화면 열기 |
| `EnemyBook add [ID]` | 특정 적 수동 등록 |
| `EnemyBook remove [ID]` | 특정 적 등록 해제 |
| `EnemyBook complete` | 모든 적을 일괄 등록 |
| `EnemyBook clear` | 도감 초기화 |

## 적 메모 태그

적 데이터베이스의 메모란에 태그를 입력하여 도감 동작을 커스터마이징할 수 있습니다.

```
<book:no>           # 이 적은 도감에 표시하지 않음
<desc1:설명 첫째 줄>
<desc2:설명 둘째 줄>
```

## 사용 예시

### 중간 보스 조우 시 강제 등록

```
[전투 처리] (보스 전투)
[플러그인 커맨드] EnemyBook add 10
[문장 표시] 강력한 적의 정보가 도감에 등록되었습니다!
```

### 튜토리얼 NPC 대화에서 도감 열기

```
[플러그인 커맨드] EnemyBook open
```

## 주의사항

- `<book:no>` 태그가 있는 적은 전투에서 만나도 등록되지 않습니다.
- 도감 데이터는 세이브 파일에 저장됩니다.
