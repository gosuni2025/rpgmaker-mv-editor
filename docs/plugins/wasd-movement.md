# WASD_Movement — WASD 이동

**파일**: `js/plugins/WASD_Movement.js`

## 기능 요약

PC 키보드의 WASD 키를 방향키로 매핑합니다. FPS/액션 게임에 익숙한 플레이어를 위한 플러그인입니다.

## 키 매핑

| WASD 키 | 대응 기능 |
|---------|----------|
| `W` | ↑ (위) |
| `A` | ← (왼쪽) |
| `S` | ↓ (아래) |
| `D` | → (오른쪽) |
| `Q` | PageUp (전 페이지) |
| `E` | PageDown (다음 페이지) |

## 파라미터

없음 (키 매핑 고정)

## 주의사항

- 기존 방향키도 그대로 사용 가능합니다 (WASD 추가, 대체 아님).
- `Q/E`가 메뉴에서 탭 전환에 사용되는 경우도 있으므로, 메뉴 조작 시 확인이 필요합니다.
- 3D 모드에서 `TouchCameraControl`과 함께 사용 시 이동과 카메라 조작이 분리됩니다.
