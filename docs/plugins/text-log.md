# TextLog — 텍스트 로그

**파일**: `js/plugins/TextLog.js`
**버전**: v1.2

## 기능 요약

게임 내 메시지 창의 대사를 자동으로 기록하여 다시 볼 수 있습니다. 메인 메뉴에서 "텍스트 로그" 항목을 선택해 과거 대화를 스크롤하며 확인할 수 있습니다.

## 주요 기능

- **자동 대사 기록** — 메시지 창 출력 내용을 자동 저장
- **선택지 기록** — 선택지 목록과 선택한 항목도 함께 기록
- **얼굴 이미지 표시** — 로그에 캐릭터 얼굴 이미지 표시 (선택)
- **세이브 연동** — 로그 기록이 세이브 파일에 자동 저장/복원
- **다양한 스크롤** — 마우스 휠, 터치 드래그, 방향키, PageUp/Down 지원

## 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| menuName | `텍스트 로그` | 메인 메뉴에 표시될 이름 |
| maxLines | `300` | 최대 로그 라인 수 (초과 시 오래된 항목 삭제) |
| entryGap | `6` | 로그 항목 사이 여백 (픽셀) |
| showFace | `true` | 얼굴 이미지 표시 여부 |
| faceSize | `100` | 얼굴 이미지 크기 (픽셀, 최대 144) |
| bgOpacity | `160` | 항목 배경 불투명도 (0~255) |
| scrollSpeed | `4` | 마우스 휠 스크롤 배율 |

## 조작 방법

| 입력 | 동작 |
|------|------|
| 마우스 휠 | 스크롤 |
| 터치 드래그 | 스크롤 |
| ↑ / ↓ 방향키 | 라인 단위 스크롤 |
| PageUp / PageDown | 페이지 단위 스크롤 |
| ESC 또는 오른쪽 클릭 | 닫기 |

## 접근 방법

플러그인 커맨드 없이 **메인 메뉴**에서 자동으로 접근할 수 있습니다:

```
메뉴 열기 → "텍스트 로그" 선택
```

## 주의사항

- 새 게임 시작 시 로그가 초기화됩니다.
- `maxLines`를 너무 높게 설정하면 세이브 파일 크기가 커질 수 있습니다.
- 비주얼 노벨 모드(`VisualNovelMode.js`)와 함께 사용하면 VN 모드 대사도 기록됩니다.
