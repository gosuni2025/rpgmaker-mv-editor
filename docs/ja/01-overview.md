# エディタ概要

## 画面構成

![エディタレイアウト](../images/overview/editor-layout.png)

エディタは 5 つのエリアで構成されています。

| エリア | 説明 |
|------|------|
| ① **メニューバー** | ファイル、編集、モード、描画、倍率、ツール、ゲーム、MCP、ヘルプ |
| ② **タブバー** | **マップ**編集タブ / **UI**編集タブの切り替え |
| ③ **ツールバー** | 編集モード · 描画ツール · 3D/FOW トグル · データベースボタン |
| ④ **サイドバー** | タイルセットパレット (上部) + マップツリー (下部) |
| ⑤ **マップキャンバス** | タイル描画 / イベント編集エリア |
| ⑥ **インスペクター** | 選択したマップ・イベント・オブジェクトのプロパティ編集 |

---

## メニューバー

### ファイル
| 項目 | ショートカット | 説明 |
|------|--------|------|
| プロジェクトを開く | — | RPG Maker MV プロジェクトフォルダを選択 |
| 保存 | `Ctrl+S` | 現在のマップを保存 |
| 新規プロジェクト | — | 新しいプロジェクトを作成 |
| 配布 | — | 配布パッケージを作成 (GitHub Pages を含む) |

### 編集
| 項目 | ショートカット | 説明 |
|------|--------|------|
| 元に戻す | `Ctrl+Z` | 直前の操作を取り消す |
| やり直す | `Ctrl+Y` | 取り消した操作をやり直す |
| 切り取り | `Ctrl+X` | 選択範囲を切り取る |
| コピー | `Ctrl+C` | 選択範囲をコピー |
| 貼り付け | `Ctrl+V` | クリップボードから貼り付け |
| 検索 | `Ctrl+F` | イベントを検索 |

### ツール
| 項目 | 説明 |
|------|------|
| データベース | アクター/スキル/アイテムなど DB 編集 |
| プラグインマネージャー | プラグインの有効化/無効化/順序変更 |
| サウンドテスト | BGM/SE 再生テスト |
| リソースマネージャー | 画像/音声ファイル管理 |
| イベント検索 | コマンド/テキストでイベントを検索 |

### ゲーム
| 項目 | ショートカット | 説明 |
|------|--------|------|
| プレイテスト | — | 現在のマップからゲームを実行 |
| 現在のマップでテスト | 右上ボタン | 現在のマップ位置から即座に開始 |

### MCP
Claude AI と連携してイベントを自動生成・修正する機能です。MCP サーバーが実行中のときに有効になります。

## ツールバー

![ツールバー](../images/overview/toolbar.png)

### 編集モード (左ボタングループ)

| ボタン | ショートカット | 説明 |
|------|--------|------|
| マップ | `F5` | タイル描画モード |
| イベント | `F6` | イベント配置/編集モード |
| ライティング | `F7` | ライティングマーカー配置モード (EXT) |
| オブジェクト | `F8` | 画像/タイルオブジェクト配置モード (EXT) |
| カメラ | `F9` | カメラゾーン編集モード (EXT) |
| 通行 | `F11` | タイル通行性編集モード |

### 描画ツール

| ツール | ショートカット | 説明 |
|------|--------|------|
| 選択 | `M` | 範囲選択 (コピー/移動) |
| 鉛筆 | `P` | 1 タイルずつ描画 |
| 消しゴム | `E` | タイル削除 |
| 塗りつぶし | — | 同じタイルをすべて置換 |
| 四角形 | — | 矩形範囲を描画 |
| 楕円 | — | 楕円範囲を描画 |
| フラッドフィル | — | 閉じた範囲を塗りつぶす |

### 表示オプション

| ボタン | 説明 |
|------|------|
| グリッド | タイルグリッド線の表示/非表示 |
| リージョン | リージョンレイヤー (z=5) の表示/非表示 |
| **3D** | 3D レンダリングモードのトグル **(EXT)** |
| ライティング | ライティングマーカーの表示/非表示 |
| **FOW** | フォグエフェクト (Fog of War) 設定 **(EXT)** |
| 縮小/拡大 | キャンバスズーム調整 |

## インスペクターパネル

右側のインスペクターは現在のマップのプロパティを表示します。

### 一般設定
- **名前** / **表示名** — マップファイル名とゲーム内表示名
- **タイルセット** — 使用するタイルセットを選択
- **スクロールタイプ** — ループなし / 横ループ / 縦ループ / 全方向ループ
- **戦闘背景** — 背景 1/2 画像
- **BGM/BGS 自動再生**

### 拡張設定 (EXT)

> `EXT` バッジがある項目は、オリジナル RPG Maker MV との互換性のために `Map*_ext.json` に別途保存されます。

| 項目 | 説明 |
|------|------|
| **3D モード** | このマップを 3D モードでプレイ |
| **3D タイルレイヤー** | 3D で上層タイルを高さとして表現 |
| **ライティングシステム** | ライティング効果を有効化 |
| **背景** | パララックス (2D 背景) またはスカイスフィア (3D スカイボックス) |
| **アニメーションタイルエディタ** | 水/滝タイルのアニメーション速度を調整 |

## キーボードショートカット

| キー | 機能 |
|----|------|
| `Ctrl+S` | 保存 |
| `Ctrl+Z` / `Ctrl+Y` | 元に戻す / やり直す |
| `F5`~`F9`, `F11` | 編集モード切り替え |
| `M` / `P` / `E` | 選択 / 鉛筆 / 消しゴム |
| `+` / `-` | ズームイン / ズームアウト |
| `マウスホイール` | ズーム調整 |
| `右クリック` ドラッグ | マップキャンバスの移動 |
| `イベントをダブルクリック` | イベントエディタを開く |
