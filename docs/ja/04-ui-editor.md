# UI エディタ

UI エディタはゲーム内の UI を視覚的に編集する機能です。
上部タブバーの **UI** タブをクリックして入ります。

UI エディタは 2 つのサブシステムで構成されています:

1. **UITheme** — ウィンドウレイアウト · 9-slice スキン · フォントカスタマイズ
2. **カスタムシーンエンジン** — JSON で新しいゲーム内 UI シーンを定義

## UITheme — ウィンドウスキンシステム

`UITheme.js` プラグインが `data/UIEditorConfig.json`、`data/UIEditorSkins.json`、`data/UIEditorFonts.json` を読み込んでゲーム UI 全体をカスタマイズします。

### ウィンドウ編集タブ

すべてのゲームウィンドウ (Window) のデフォルトプロパティを設定します。

**設定可能な項目**:
- デフォルト windowskin 画像
- デフォルトフォントサイズ / 色 / アウトライン
- 背景透明度 (backOpacity)
- カラートーン (colorTone: R/G/B)

### フレーム編集タブ

ゲーム内 UI ウィンドウでよく使用されるボーダー (フレーム) を視覚的に編集します。windowskin 画像から背景とボーダー領域を指定し、9-slice 方式でどのサイズのウィンドウにも自然に伸びるよう設定します。

![9-slice スキンエディタ](../images/ui-editor/skin-editor.png)

- **左**: 登録済みスキンリスト — ゲージ (HP/MP/TP)、カスタムスキンなどを個別管理
- **中央キャンバス**: windowskin 画像上でフレーム領域と背景領域をドラッグで直接指定
- **右インスペクター**: フレーム領域 (X/Y/幅/高さ)、フィル領域 (背景)、コーナーサイズを数値で調整
- **プレビュー**: 実際の適用結果をリアルタイムで確認

### カーソル編集タブ

選択カーソルの画像とアニメーションを設定します。

### フォント設定タブ

フォントプリセットを定義します。各ウィンドウに異なるフォントを適用できます。

- フォントファイル: `fonts/` フォルダの TTF/WOFF ファイル
- フォントサイズ
- アウトライン色/幅
- 行の高さ

## カスタムシーンエンジン

`CustomSceneEngine.js` プラグインが `data/UIScenes/_index.json` + シーン別 JSON ファイルを読み込んでゲーム内 UI シーンを動的に生成します。

### シーン選択と作成

左上の**シーン選択ドロップダウン**から編集するシーンを選択します。

`+` ボタンで新しいシーンを追加します。

### シーンの置き換え

インスペクターの**シーン置き換え**オプション:
- **ゲーム内デフォルトシーンを使用** — オリジナルの RPG Maker MV シーンを使用
- **カスタムシーンに置き換え** — `data/UIScenes/` で定義されたシーンに置き換え

例: `Scene_Options` をカスタムシーンに置き換えると、ゲーム内のオプション画面が完全に新しいレイアウトに置き換えられます。

### ウィジェットツリー (formatVersion 2)

カスタムシーンはウィジェットツリーで構成されます。左側のシーンリストは階層的なウィジェット構造を表示します。

**ウィジェットの種類**:

| ウィジェット | 説明 |
|------|------|
| `Panel` | レイアウトコンテナ (子ウィジェットを配置) |
| `List` | 選択可能なコマンドリスト |
| `Label` | テキスト表示 |
| `Image` | 画像表示 |
| `ActorFace` | アクター顔画像 |
| `Gauge` | HP/MP/EXP ゲージ |
| `Button` | クリック可能なボタン |

### テンプレート変数

テキストに `{変数}` 形式で動的な値を挿入します:

```
{actor[0].name}    → 最初のパーティーメンバーの名前
{actor[0].level}   → 最初のパーティーメンバーのレベル
{actor[0].hp}      → 最初のパーティーメンバーの HP
{var:10}           → ゲーム変数 #10 の値
{switch:5}         → ゲームスイッチ #5 の状態 (true/false)
{gold}             → パーティーの所持金
{config.bgmVolume} → 設定値 (bgmVolume)
```

### サンプルシーンの編集

![サンプルタイトルシーン編集](../images/ui-editor/ui-title-navigation.png)

エディターにはタイトル、オプション、ステータスなどよく使うシーンがサンプルとして含まれています。最初から作る必要はなく、サンプルシーンを選択してボタンの位置・テキスト・背景などを修正するだけで、簡単にオリジナルの UI を完成させることができます。

**Nav Visual** モードをオンにすると、ボタン間のナビゲーション接続を視覚的に確認・編集できます。青/オレンジの線は方向キー入力時にフォーカスが移動するパスを示します。

---

### リアルタイムテスト

**現在のマップでテスト**ボタンで、現在の UI 設定をゲームで即座に確認できます。

シーン編集ウィンドウ上部の**入場** / **退場**ボタンで、シーントランジションアニメーションをプレビューできます。

## UIEditorConfig.json の構造

```json
{
  "windows": [
    {
      "windowClass": "Window_Options",
      "windowStyle": "frame",
      "windowskinName": "MyCustomSkin",
      "x": 100, "y": 50,
      "width": 400, "height": 300,
      "fontSize": 20,
      "backOpacity": 180,
      "colorTone": [0, 0, 30, 0]
    }
  ]
}
```

---

## UIEditorSkins.json の構造

```json
{
  "skins": [
    {
      "name": "MyCustomSkin",
      "windowskin": "Window_Blue",
      "fillArea": { "x": 0, "y": 0, "w": 64, "h": 64 },
      "frameInfo": { "x": 0, "y": 64, "w": 64, "h": 64, "cornerSize": 12 }
    }
  ]
}
```

---

## UIEditorFonts.json の構造

```json
{
  "fonts": [
    {
      "name": "GameFont",
      "face": "dotgothic16",
      "size": 22,
      "outlineColor": "rgba(0,0,0,0.7)",
      "outlineWidth": 3
    }
  ]
}
```
