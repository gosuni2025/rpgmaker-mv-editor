# マップエディタ

## マップキャンバス

![Big City 2D マップ](../images/map-editor/big-city-2d.png)

マップキャンバスはエディタの中心です。タイルパレットで選択したタイルをキャンバスに描画したり、イベントを配置・編集したりできます。

---

## サイドバー

### タイルセットパレット

サイドバーの上部にはタイルセットパレットがあります。

**タブ構成**:
- `A` — A1〜A5 オートタイル (水、地形、建物外壁、壁)
- `B` — B タイルセット
- `C` — C タイルセット
- `D` — D タイルセット
- `R` — リージョンレイヤー (Region ID 1〜255)

パレットでタイルをクリックして選択し、ドラッグで複数のタイルを一度に選択できます。

### マップツリー

サイドバーの下部にはマップツリーがあります。
- マップを**ダブルクリック**するとそのマップが開きます。
- マップを**右クリック**すると新規追加/削除/プロパティ編集メニューが表示されます。
- マップをドラッグして階層構造を変更できます。

---

## 編集モード

### マップ編集モード (`F5`)

タイルを描画するモードです。上部ツールバーで描画ツールを選択します。

**レイヤー選択**:
![タイルセットパレット](../images/map-editor/tileset-palette.png)

**描画ツール**:

| ツール | 操作 |
|------|------|
| 鉛筆 (P) | クリック/ドラッグで 1 タイルずつ描画 |
| 選択 (M) | ドラッグで範囲選択 → コピー (Ctrl+C)/移動 |
| 消しゴム (E) | クリック/ドラッグでタイルを削除 |
| 塗りつぶし | クリックした位置から同じタイルをすべて置換 |
| 四角形 | ドラッグで矩形範囲を描画 |
| 楕円 | ドラッグで楕円範囲を描画 |
| フラッドフィル | 閉じた範囲の内部を塗りつぶす |

**タイル情報ツールチップ**: マップ上にマウスをかざすと、そのタイルのレイヤー別情報が表示されます。

### イベント編集モード (`F6`)

イベントを配置・編集するモードです。

- **空のタイルをダブルクリック** — 新規イベントを作成
- **イベントをダブルクリック** — イベントエディタを開く
- **イベントを右クリック** — コピー/貼り付け/削除メニュー
- **イベントをドラッグ** — 位置を移動

イベントの上にはイベント ID とキャラクタースプライト (ある場合) が表示されます。

### 通行性編集モード (`F11`)

タイルごとの通行性 (移動可否) を編集します。

- ○ — 通行可能
- × — 通行不可
- ↑↓←→ — 方向別通行性

## ライティングシステム (EXT)

> マップインスペクターで**ライティングシステム**チェックボックスを有効にする必要があります。

### ライティングモード (`F7`)

ライティングマーカーをマップに配置・編集します。

**ライトの種類**:
- **ポイントライト** — 特定位置から全方向に光を発する
- **アンビエントライト** — マップ全体の基本輝度
- **スポットライト** — 方向性のあるライティング

**ライティングマーカー**: 編集モードでは光の形のマーカーとして表示されます。

**インスペクターで編集**:
- 色 (RGB)
- 強度 (Intensity)
- 範囲 (Range)
- 減衰方式

Lava Cave の例 — ライティング効果:

![Lava Cave 3D ライティング](../images/3d-mode/lava-cave-lighting.png)

## オブジェクトシステム (EXT)

![オブジェクトモード](../images/map-editor/object-mode.png)

マップに画像/タイルオブジェクトを自由に配置するシステムです。

### オブジェクトモード (`F8`)

左側のオブジェクトリストからオブジェクトを選択するか、新規作成できます。

**オブジェクトの種類**:
- **タイルオブジェクト** — タイルセットから選択したタイルを配置 (整数倍サイズ)
- **アウトラインオブジェクト** — マップのタイル範囲を指定
- **画像オブジェクト** — 画像ファイルを直接配置 (任意サイズ/位置)
- **アニメーションオブジェクト** — データベースのアニメーションを再生

**オブジェクトプロパティ** (インスペクター):
- 位置 (X, Y, Z)
- サイズ (幅 N タイル)
- 高さ (Z 軸) オフセット
- レンダーレイヤー順序

## カメラゾーン (EXT)

カメラゾーンを使用すると、特定のマップエリアに進入したときにカメラの位置・角度が自動的に変化します。

### カメラモード (`F9`)

- キャンバスをドラッグしてカメラゾーンエリアを作成
- エリアをクリックして選択し、インスペクターで設定を編集

**カメラゾーン設定**:
- **ターゲット位置** — カメラが見る中心点
- **カメラオフセット** — カメラ位置の調整
- **ズーム** — ズーム倍率
- **イージング** — トランジションアニメーション方式

---

## FOW (Fog of War) (EXT)

ツールバーの **FOW** ボタンでフォグエフェクトを設定します。

**FOW 設定** (インスペクター):
- **有効** — FOW のオン/オフ
- **視野半径** — 探索可能範囲 (タイル単位)
- **フォグ色** — 未探索エリアの色
- **フォグ不透明度**
- **エディターで表示** — エディター画面で FOW をプレビュー

---

## マッププロパティ

マップツリーでマップを**右クリック → プロパティ**、またはマップを開いてインスペクターで直接編集します。

**設定可能な項目**:
- 名前 / 表示名
- マップサイズ (横 × 縦タイル数)
- タイルセット
- スクロールタイプ (ループ設定)
- 戦闘背景
- BGM/BGS 自動再生
- **3D モード** (EXT)
- **スカイボックス** (EXT)
- **ライティングシステム** (EXT)
- **FOW** (EXT)
- メモ

---

## アニメーションタイルエディタ (EXT)

インスペクター下部の**アニメーションタイルエディタ**セクションで、水/滝オートタイルのアニメーション設定を調整します。

- **Water A (Meadow)** — 草原の水アニメーション
- **Water B (Snow)** — 雪上の水アニメーション
- **Waterfall** — 滝アニメーション
- **Canal** — 水路アニメーション

各タイルの**速度** (フレーム/ティック) と**シェーダー適用**の有無を設定します。
